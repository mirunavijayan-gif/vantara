# 🚗 VanTara - On Road Vehicle Assistance App

## 📌 Project Description

VanTara is an Android application designed to provide quick and reliable on-road vehicle assistance.
It connects users with nearby mechanics and allows administrators to manage the entire system.

---

## 🎯 Features

### 👤 User Module

* Register and Login
* Request vehicle assistance
* View service status

### 🔧 Mechanic Module

* Login to dashboard
* Set availability (Available/Busy)
* View service requests
* Accept or reject requests

### 🛠️ Admin Module

* Manage users
* Manage service providers (mechanics)
* Handle complaints
* Manage payments
* Generate reports

2. Click on "Open Project"
3. Select the project folder
4. Wait for Gradle build to finish
5. Click Run ▶️
